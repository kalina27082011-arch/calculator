# 💰 Калькулятор семейного бюджета

Простой веб-приложение для планирования семейного бюджета: учёт доходов и расходов, расчёт накоплений, цель сбережений и визуализация структуры бюджета.

Работает полностью в браузере — без сервера и регистрации. Данные сохраняются локально.

## Возможности

- **Доходы и расходы** — добавление и удаление статей
- **Типы расходов** — обязательные и переменные
- **Категории** — жильё, еда, транспорт, развлечения, прочее
- **Цель накоплений** — процент от дохода с подсказкой о прогрессе
- **Диаграмма** — наглядная структура бюджета
- **Автосохранение** — данные хранятся в `localStorage`
- **Экспорт / импорт** — резервная копия в JSON
- **Печать** — версия для печати отчёта
- **Тёмная и светлая тема**

## Быстрый старт

1. Склонируйте или скачайте репозиторий
2. Откройте `index.html` в браузере (или используйте расширение Live Server в VS Code / Cursor)
3. Нажмите «Открыть калькулятор» и начните вводить данные

## Публикация (GitHub Pages)

1. Загрузите проект на GitHub
2. В настройках репозитория: **Settings → Pages**
3. Source: ветка `main`, папка `/ (root)`
4. Сайт будет доступен по адресу `https://<username>.github.io/<repo>/`
